# Forkify App

A modern JavaScript application for searching and viewing recipes.

## ⚠️ Note

The recipe upload feature is not implemented due to issues with generating an API key from the Forkify API service.

Despite multiple attempts over several days, the API consistently returned a "try again later" response, which prevented further development of this functionality.

All other core features of the application are fully implemented, including:

- Recipe loading and display
- Search functionality
- Pagination
- MVC architecture

This project demonstrates a complete front-end application structure and asynchronous data handling using modern JavaScript.
